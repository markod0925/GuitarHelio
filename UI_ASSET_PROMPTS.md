# UI Asset Prompts (Neon Blue Theme)

Use these prompts to generate assets aligned with the current GuitarHelio UI style.
All assets should be exported as PNG with transparent background unless otherwise noted.

## 1) Main Logo
- Filename: `public/ui/logo-guitarhelio-neon.png`
- Suggested size: `1024x320`
- Prompt:
  `Create a clean game logo that reads "GuitarHelio", metallic silver-blue letters, subtle futuristic bevel, cool blue rim light, soft cyan outer glow, high contrast, minimal ornaments, premium casual game style, transparent background, no watermark, no extra text`

## 2) Song Cover Placeholder
- Filename: `public/ui/song-cover-placeholder-neon.png`
- Suggested size: `512x512`
- Prompt:
  `Create a square placeholder album art icon for a rhythm game, dark navy gradient background, soft vignette, centered musical note glyph, glassmorphism and neon blue highlights, subtle glow, minimal and elegant, transparent outer background not needed, no text`

## 3) Featured Cover (Space / Epic Theme)
- Filename: `public/ui/song-cover-featured-space.png`

## 8) Horizontal Neon String Line Tile (Optional)
- Filename: `public/ui/neon-line-tile.png`
- Suggested size: `1024x64`
- Prompt:
  `Create a seamless horizontal neon line tile for a rhythm game background, dark navy base with one glowing electric blue line and tiny bloom, minimal noise, seamless left-right tiling, no text`

## Output Guidelines
- Keep contrast high on dark blue backgrounds.
- Avoid purple dominance.
- Avoid tiny details that disappear at small UI sizes.
- Export 1x and 2x versions if possible.
